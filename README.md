  노트정리 : 인턴 활동 중 공부하면서 작성한 노트필기
  인턴활동 : 작성한 QA문서 및 코드

</pre>

## 인턴활동/개발활동
<pre>
  제가 작성했던 코드가 들어있는 폴더를 가져왔습니다.
  admin - 전부 제가 작성했습니다.
  나머지 각 폴더들은 admin을 작성하기 위해 제가 작성했던 부분이 들어있는 부분만 가져왔습니다.
</pre>
